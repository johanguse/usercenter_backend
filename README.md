# Project Management API

## Description

This is a project management API that allows you to create, read, update, and delete projects, teams, and team members.

## Technologies

- FastAPI
- FastAPI Users
- FastAPI Security
- FastAPI Pagination
- SQLAlchemy
- Alembic
- PostgreSQL
- Uvicorn
- Pytest
- Poetry
- Ruff


## How to run

1. Clone the repository
2. Run `poetry shell`
3. Run `poetry install`
4. Run `task run`

## How to test

Run `task test`

## How to format

Run `task format`

## How to lint

Run `task lint`
